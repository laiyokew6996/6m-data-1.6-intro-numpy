Part 4 — The Hidden Wire

A slice shares memory. A copy does not.

This is the #1 NumPy surprise. A basic slice gives you a view — a second name for the same data. Edit the view, and the original changes with it. Use .copy() when you want a safe, independent version.

Scenario: You take 3 days of sales out of the week to "adjust them for a promo". Will the original week's record change?
week = np.array([100, 120, 95, 140, 160, 200, 90])
three_days = week[1:4]
